Blueprint Pattern

One prompt to adopt. Checkout + prompt for every session after.

  1. 1
    Run the adoption prompt

    Open your app repo in the IDE. New chat → paste prompt. The agent downloads templates, writes always-on.md, creates blueprint.md, and starts documentation.

    Manual zip if the agent cannot run shell commands.

  2. 2
    Follow-up sessions

    Work through blueprint.md. Repeat until bootstrap is complete.

  3. 3
    Close creation

    Review session (new chat, report-only).

Pick a mode. Copy checkout → terminal. Copy prompt → new agent chat.
