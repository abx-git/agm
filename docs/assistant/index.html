Blueprint Pattern

One prompt to adopt. Copy a session prompt for every chat after.

  1. 1
    Run the adoption prompt

    Open your app repo in the IDE. New chat → paste prompt. The agent writes the scaffold, always-on.md, blueprint.md, and the first documentation section — no download, no checkout script.

  2. 2
    Follow-up sessions

    Work through blueprint.md. New chat → copy prompt. Repeat until bootstrap is complete.

  3. 3
    Close creation

    Review session (new chat, report-only).

Pick a mode. New chat → copy prompt → paste.
