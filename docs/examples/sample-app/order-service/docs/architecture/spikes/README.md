# Architecture Spikes — order-service

Outputs from **Spike** sessions (formerly Architecture Work / WRK). Each spike is a **folder** under `spikes/` with notes, Mermaid, and optional E2 boards.

## Legacy flat work/

Older items remain as single Markdown files in [`../work/`](../work/) with `WRK-*` ids. New explorations use **SPK-*** under `spikes/`.

## New spike

1. In **AGM Studio → Spike → New**, or copy [`../../../../../../agm/scaffold/doc-root/spikes/_template/`](../../../../../../agm/scaffold/doc-root/spikes/_template/)
2. Register in `blueprint.md` ## Spike register
3. Optionally attach boards via Studio or E2 (`.storm.json` in `boards/`)

See [Guide](../../../../../../guide.md).
